
## 🚀 Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/jta-ads/portifolio.git
   ```
2. Acesse a pasta:
   ```bash
   cd portifolio
   ```
3. Abra o arquivo `index.html` no navegador.
